# JS-unfollow-IG-max-200-per-day-

- เปิด "กำลังติดตาม" หรือ "Following" จะแสดงป็อปอัพขึ้นมา
- คลิกขวา Inspect 
- ไปที่ช่อง Console
- วางโค้ดคำสั่ง 
- กด Enter

ปล.IG จำกัดการเลิกติดตามที่ 200 Account ต่อวัน

---

- Open Ig profile on Website
- Open "Following"
- Right Click > Inspect 
- Go to Console Panel
- Paste code
- Press Enter
